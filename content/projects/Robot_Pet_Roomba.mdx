## Robot Pet Project Status: TBD

Roomba 960, Raspberry Pi 5, Llama 3.1 70B, and zero monthly cost after hardware.

---

## Overview

A Roomba 960 converted into a voice controlled AI companion. It wanders the house, sees through a webcam, hears you speak, and responds with personality. Audio and video stay inside the house. Only transcribed conversation text is sent to the cloud.

The goal was to build something more physical and more playful than a chatbot on a laptop. This project brings together speech, vision, memory, and movement in a single system built from relatively accessible hardware.

## Build Phases

| Phase | Description |
|---|---|
| 01 | Verify the Roomba with a full stock cleaning cycle and confirm navigation works normally |
| 02 | Set up the Pi 5 with NVMe boot, SSH, and the serial connection to the Roomba |
| 03 | Build the voice loop so whisper hears, Groq responds, and Piper speaks |
| 04 | Add vision so Moondream can describe the room and provide context |
| 05 | Parse action tags from the model response and translate them into movement commands |
| 06 | Store conversations and learned facts in SQLite so the robot can retain memory across sessions |
| 07 | Mount the hardware on the Roomba and run the system untethered |
| 08 | Tune the robot personality, name, and response style |
