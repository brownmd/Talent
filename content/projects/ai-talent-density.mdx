## Overview


1. Identifies AI-relevant GitHub activity from GitHub Archive.
2. Scores GitHub users monthly activity based on event type and repo category.
3. Enriches GitHub users with public GitHub suer profile locations and normalizes those locations.
4. Creates ai user talent density visualizations, insights and reports.

## Scoring Model

- ReleaseEvent: 30
- PullRequestReviewEvent: 25
- DeploymentEvent: 25
- PullRequestEvent: 20
- CreateEvent: 15
- CheckRunEvent: 15
- PushEvent: 10
- WorkflowRunEvent: 10
- MemberEvent: 10
- IssuesEvent: 5
- IssueCommentEvent: 4
- ForkEvent: 2
- WatchEvent: 1

### Repo multipliers

GitHub Copilot suggested this... 

- 5.0x: scaling, interpretability, RL, frontier infra
- 3.5x: agentic orchestration and MCP ecosystems
- 2.0x: SDKs, frameworks, and builder tools
- 1.5x: local tooling and AI assisted development tools
- 1.0x: all other qualifying AI activity
