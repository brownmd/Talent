## Overview

This is a monthly-updated analytics pipeline with production guardrails and stable consumer-facing views — not a one-off heatmap.

The system ingests vast volumes of public GitHub activity, applies sophisticated scoring and enrichment, and surfaces insights through Metabase dashboards, geospatial maps (Kepler.gl), and Plotly exports.

## Signals Used to Score AI Literacy

| Signal | Points |
|---|---|
| Contributor to AI repo | 10 pts |
| Works at AI company | 6 pts |
| Forks AI repo | 5 pts |
| Bio mentions AI / ML / LLM | 3 pts |
| Primary language Python or R | 4 pts |
| Stars AI repo | 2 pts |

**AI Literacy score formula:**

$$
\text{Score}_{county} = \frac{\sum_{i=1}^{n} w_i \cdot s_i}{\text{population}} \times 10{,}000
$$

Where $w_i$ is the point weight for signal $i$, and $s_i$ is the number of users in that county exhibiting that signal.

## Tech Stack

| Layer | Technology |
|---|---|
| Event Data | GitHub Archive |
| Query Engine | BigQuery |
| Population Data | US Census API |
| Database | PostgreSQL 15 + PostGIS (Hetzner VPS) |
| Backend / ETL | Python |
| Geocoding | geopy / Nominatim + GeoNames |
| Visualization | Metabase, Kepler.gl, Plotly |
| Infrastructure | Hetzner VPS (Ubuntu, Docker Compose) |
| Access & Delivery | Tailscale, Cloudflare |
